# Weather
WeatherApp is a simple single-page application for displaying weather information.
## Features
- ### Geolocation:
Allow access to your current location to display the weather at your location.
- ### City Search: 
Enter the name of a city in the search bar to get weather data for the selected location.
- ### Visual Elements: 
Unique visual elements corresponding to the current weather conditions.
- ### Detailed Information: 
Includes data on temperature, feels-like temperature, and precipitation.
## Installation
Just clone the repository:
```
git clone https://github.com/white4ocolate/Weather-Forecast.git
```
## Usage
1. Open Weather.xcodeproj in Xcode.
2. Run the project on a simulator or a real device.
3. Allow access to your location to get weather data for your current location, or use the search bar to select a city.

## Requirements
- iOS 17.0+
- Xcode 15.0+
- Swift 5.0+
